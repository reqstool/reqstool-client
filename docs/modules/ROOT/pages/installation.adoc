= Installation

== Prerequisites

* Python 3.10 or later

== Install with pipx (recommended)

https://pipx.pypa.io/[pipx] installs reqstool in an isolated environment and makes it available globally:

[source,bash]
----
pipx install reqstool
reqstool -h  # confirm installation
----

== Install with pip

Alternatively, install with pip:

[source,bash]
----
pip install reqstool
reqstool -h  # confirm installation
----
